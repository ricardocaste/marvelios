# marvelios
Marvel App made with SwiftUI, MVVM, Combine

## Features:
- MVVM
- Combine
- SwiftUI
- Clean Architecture
- Realtime Localization
- Theme Switcher
- Repository and Dependency Injection Design Patterns
- Unit Testing: Network Layer, CoreData, Repository, JSON Decodable
